Lesson 01 — OpenUSD 基礎概念

來源：openusd.org 官方文檔 v26.05

什麼是 USD？

USD (Universal Scene Description) 是 Pixar 開發的開源 3D 場景描述框架，用於：

  • 交換 3D 資產（模型、動畫、材質）
  • 組裝 多個資產到虛擬場景中
  • 非破壞性編輯 場景（override 機制）
  • 跨應用協作 — Maya / Houdini / Blender / UE5 / Omniverse 共用同一場景

五大核心概念

1. Stage（場景圖）

Stage = 根 USD 檔案 + 所有它引用/疊加的圖層 → 合成後的完整場景
  • 對應類別：UsdStage
  • 負責：管理命名空間、快取、記憶體中的場景遍歷
  • UsdStage::Save() 把修改寫回檔案

2. Prim（基本容器）

Prim = USD 場景樹中的節點
  • 每個 Prim 可以有：
  • 子 Prim（形成命名空間階層）
  • Attributes（資料，如位置、顏色）
  • Relationships（指向其他 Prim）
  • Metadata（如 kind、active 狀態）
  • 對應類別：UsdPrim
  • Prim 有 Specifier（def / over / class）定義角色
  • Prim 可能有 schema typeName 決定資料結構
# USD ASCII 範例
def Xform "World" {
    def Sphere "BigBall" {
        double radius = 100
    }
}

→ 路徑：/World/BigBall

3. Layer（圖層）

Layer = USD 的最小持久化單位（一個檔案）
  • 對應類別：SdfLayer
  • 檔案格式：
  • .usda — 文字格式（human-readable ASCII）
  • .usdc — 二進位格式（高效、預設）
  • .usd — 可以是文字或二進位
  • .usdz — 打包格式（zip，含材質貼圖）
  • Layer 之間的關係由 Composition Arcs 定義

4. Attribute（屬性）

Attribute = 具型別的屬性，存儲在 Prim 上
  • 每個 Attribute 有：
  • 型別（double, float3, token, string, asset...）
  • 預設值（default）
  • 動畫值（timeSamples：時間→值的映射）
def Sphere "BigBall" {
    double radius = 100             # default value
    double radius.timeSamples = {   # animated values
        1: 100,
        24: 500,
    }
}

→ frame 1 時 radius=100，frame 24 時 radius=500，其餘內插

5. Composition（合成）

Composition = 把多個 Layer 依照 Composition Arcs 合併成一個 Stage 的過程

四種 Composition Arcs（由弱到強）：

Arc用途指令
SubLayers多藝術家同時編輯同場景subLayers = [@layer1.usd@, @layer2.usd@]
References引用外部資產references = @asset.usd@
Payloads延遲載入（優化效能）payload = @heavy_asset.usd@
Inherits繼承 class 定義inherits =
Specializes特化（最強限制）specializes =
VariantSets變體切換（如 LOD）variantSet "lod" = { "high" ... "low" ... }

LIVRPS 強度順序（弱→強）：

Local > Inherits > Variants > References > Payloads > Specializes > SubLayers

越右邊的意見越強，會覆蓋左邊的同名屬性。

檔案結構 vs 場景結構

# 檔案系統
assets/
├── room.usd          ← 根 Layer（定義 subLayers）
├── furniture.usd     ← 被 subLayer 疊加
└── chair.usd         ← 被 reference 引用

# room.usd 內容
#usda 1.0
(
    subLayers = [@furniture.usd@]
)
def Xform "Room" {
    def "Chair1" (references = @chair.usd@) {}
    def "Chair2" (references = @chair.usd@) {}
}

開啟 room.usd 得到 Stage：

/Room
/Room/Chair1    ← 從 chair.usd 合成
/Room/Chair2    ← 從 chair.usd 合成（另一實例）
/Furniture/...  ← 從 furniture.usd 疊加

Python API 快速入門

from pxr import Usd, UsdGeom, Sdf

# 1. 建立 Stage
stage = Usd.Stage.CreateNew("hello.usda")

# 2. 建立 Prim（Xform = Transform 容器）
xform = UsdGeom.Xform.Define(stage, "/World")

# 3. 建立 Sphere Prim + 設屬性
sphere = UsdGeom.Sphere.Define(stage, "/World/MySphere")
sphere.GetRadiusAttr().Set(50.0)

# 4. 儲存
stage.GetRootLayer().Save()

# 5. 開啟已有 Stage
stage = Usd.Stage.Open("hello.usda")

# 6. 遍歷所有 Prim
for prim in stage.Traverse():
    print(prim.GetPath())  # /World, /World/MySphere

# 7. 讀取屬性
sphere = stage.GetPrimAtPath("/World/MySphere")
radius = sphere.GetAttribute("radius").Get()
print(radius)  # 50.0
