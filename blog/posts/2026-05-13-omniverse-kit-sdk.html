Lesson 03 — Omniverse 入門：Kit SDK 與 Extension 架構

涵蓋 Kit SDK 核心概念 + Extension 開發基礎

來源：docs.omniverse.nvidia.com/kit/docs/kit-manual v110.1.0

什麼是 Omniverse Kit？

Omniverse Kit 是 NVIDIA 提供的 SDK，用來建立 Omniverse 應用程式（如 USD Composer、USD Explorer）。核心元件：

Omniverse Kit
├── USD / Hydra       ← 場景描述 + 渲染管線
├── Omni.USD          ← C++ 層，橋接 USD + Kit Core + OmniClient
├── Carbonite          ← 外掛框架（plugin system）
├── RTX Renderer       ← 即時光線追蹤渲染
├── omni.ui            ← UI 工具包（類似 Qt/PySide）
├── omni.client        ← 連線 Omniverse Nucleus 伺服器
└── Python Scripting   ← 所有功能都有 Python binding

核心概念：Extension（擴展）

Extension 是什麼？

Kit 的所有功能都以 Extension 形式存在。一個 Extension 可以：

  • 載入 Python 模組
  • 載入 C++ / Carbonite 外掛
  • 啟用其他 Extension（依賴）
  • 套用設定（settings）
  • 註冊 UI、命令、事件

目錄結構

extensions/
└── omni.example.hello/          ← Extension 根目錄
    ├── config/
    │   └── extension.toml        ← Extension 定義檔（必需）
    ├── omni/
    │   └── example/
    │       └── hello/
    │           ├── __init__.py   ← Python 入口
    │           └── window.py     ← 視窗邏輯
    ├── data/                     ← 靜態資源
    ├── docs/                     ← 文件
    └── bin/                      ← 二進位檔案（如有 C++）

Extension ID

omni.example.hello-1.0.0
│                 │
└── ext_name      └── version (semver)
  • ext_name：命名空間.類別.名稱（如 omni.kit.commands）
  • version：semver（major.minor.patch-prerelease）

extension.toml 完整範例

# ── 基本資訊 ────────────────────────────
[package]
version = "1.0.0"
title = "Hello World Extension"
description = "My first Omniverse extension"
category = "example"
authors = ["Your Name <email@example.com>"]

# ── 依賴 ────────────────────────────────
[dependencies]
"omni.kit.commands" = {}        # 無版本限制
"omni.usd" = "1.0.0"            # 最低版本
"omni.ui" = "2.0"

# 可選依賴
[optional_dependencies]
"omni.kit.window.filepicker" = {}

# ── Python 模組 ──────────────────────────
[python]
# 載入的 Python 模組（對應 omni/example/hello/__init__.py）
modules = [
    "omni.example.hello"
]

# ── Carbonite 外掛（如有 C++）────────────
[[plugin]]
path = "bin/windows-x86_64/release/omni.example.hello.plugin.dll"

# ── 設定 ────────────────────────────────
[settings]
app/exts/omni.example.hello/enabled = true
app/exts/omni.example.hello/windowTitle = "Hello World"

# ── Extension 可能觸發的事件 ─────────────
[events]
"omni.kit.commands.execute" = "on_command_executed"

Python Extension 最小範例

omni/example/hello/__init__.py

import omni.ext
import omni.ui as ui

class HelloExtension(omni.ext.IExt):
    """Extension 生命週期管理"""

    def on_startup(self, ext_id):
        """Extension 被啟用時呼叫"""
        print(f"[{ext_id}] Hello Extension starting up")
        self._window = ui.Window("Hello World", width=300, height=200)
        with self._window.frame:
            with ui.VStack():
                ui.Label("Welcome to Omniverse Kit!")
                ui.Button("Click Me", clicked_fn=self._on_click)

    def on_shutdown(self):
        """Extension 被停用時呼叫"""
        print("Hello Extension shutting down")
        if self._window:
            self._window.destroy()
            self._window = None

    def _on_click(self):
        print("Button clicked!")

Extension 生命週期

載入階段:
  1. Kit 掃描 extension 搜尋路徑
  2. 讀取 extension.toml
  3. 解析依賴（dependencies）
  4. 依序載入依賴 Extension

啟用階段:
  5. 載入 [python].modules 指定的 Python 模組
  6. 呼叫 on_startup(ext_id)
  
執行階段:
  7. Extension 回應事件、建立 UI、操作 USD

停用階段:
  8. 呼叫 on_shutdown()
  9. 釋放資源

常用 API

omni.usd — USD 操作

import omni.usd

# 取得目前 Stage
stage = omni.usd.get_context().get_stage()

# 監聽 Stage 變化
def on_stage_opened(result, error):
    if result:
        print("Stage opened!")

omni.usd.get_context().open_stage_async("path/to/scene.usda", on_stage_opened)

omni.kit.commands — 命令系統

import omni.kit.commands

# 執行內建命令
omni.kit.commands.execute("CreatePrim", prim_type="Sphere", prim_path="/World/MySphere")

# 註冊自訂命令
omni.kit.commands.register("MyCustomCommand", my_execute_fn)

# 復原
omni.kit.undo.begin_group()
omni.kit.commands.execute("CreatePrim", prim_type="Cube", prim_path="/World/Cube1")
omni.kit.commands.execute("CreatePrim", prim_type="Cube", prim_path="/World/Cube2")
omni.kit.undo.end_group()
omni.kit.undo.undo()  # 復原兩個 Cube

omni.ui — UI 工具包

import omni.ui as ui

# 建立視窗
window = ui.Window("My Tool", width=400, height=300)
with window.frame:
    with ui.VStack(spacing=5):
        ui.Label("Settings", style={"font_size": 18})
        with ui.HStack():
            ui.Label("Radius:")
            radius_field = ui.FloatField()
            radius_field.model.set_value(1.0)
        ui.Button("Apply", clicked_fn=lambda: print("Applied!"))

omni.timeline — 時間軸

import omni.timeline

timeline = omni.timeline.get_timeline_interface()
timeline.play()
timeline.pause()
timeline.set_current_time(100)  # frame 100
print(timeline.get_current_time())

Extension 搜尋路徑

Extension 可以放在這些位置：

路徑用途
{app}/exts/應用內建 Extension
{app}/extsPhysics/物理 Extension
$HOME/Documents/Kit/extensions/使用者自訂
環境變數 OMNI_EXTS_PATHS額外搜尋路徑
Registry (Nucleus)從 Omniverse 伺服器下載

實際開發流程

# 1. 建立 Extension 目錄
mkdir -p ~/Documents/Kit/extensions/omni.example.mytool/config
mkdir -p ~/Documents/Kit/extensions/omni.example.mytool/omni/example/mytool

# 2. 建立 extension.toml
cat > ~/Documents/Kit/extensions/omni.example.mytool/config/extension.toml << 'EOF'
[package]
version = "1.0.0"
title = "My Tool"
category = "example"

[dependencies]
"omni.usd" = {}
"omni.ui" = {}

[python]
modules = ["omni.example.mytool"]
EOF

# 3. 建立 __init__.py
cat > ~/Documents/Kit/extensions/omni.example.mytool/omni/example/mytool/__init__.py << 'EOF'
import omni.ext
import omni.ui as ui

class MyToolExtension(omni.ext.IExt):
    def on_startup(self, ext_id):
        self._window = ui.Window("My Tool", width=300, height=200)
        with self._window.frame:
            with ui.VStack():
                ui.Label("Hello from My Tool!")

    def on_shutdown(self):
        if self._window:
            self._window.destroy()
EOF

# 4. 啟動 Kit 應用，Window → Extensions → 搜尋 "My Tool" → 啟用

Extension 類型

類型說明
Python-only只有 Python 程式碼，最常見
C++ Plugin有 bin/ 下的 native 外掛
Single-file只有 extension.toml，純設定/依賴
App Extension[package.app] = true，可獨立啟動
Feature Extension[package.feature] = true，在使用者 UI 中顯示
USD Schema定義自訂 USD Schema（下一課）
