什麼是 Isaac Sim？

基於 Omniverse Kit 建立，專門用於：

  • 機器人運動學/動力學模擬
  • 感測器模擬（相機、LiDAR、IMU）
  • ROS/ROS2 整合
  • Synthetic Data Generation（合成資料）
  • Reinforcement Learning 訓練環境
Isaac Sim
├── Omniverse Kit (USD + RTX + UI)
├── PhysX 5 (物理引擎)
├── ROS/ROS2 Bridge
├── Sensor Simulation (RTX-based)
├── Cortex (高階機器人控制)
└── Python API (omni.isaac.*)

ROS/ROS2 Bridge

Isaac Sim 內建 ROS2 Bridge，可與真實 ROS 系統雙向通訊：

┌──────────────┐           ┌──────────────┐
│  Isaac Sim   │ ←─ROS2──→ │  ROS2 Node   │
│  (模擬環境)   │   topics  │  (控制演算法) │
└──────────────┘           └──────────────┘

啟動 ROS2 Bridge

from omni.isaac.ros2_bridge import Ros2Bridge

# 建立 Bridge
bridge = Ros2Bridge()

# 發布 joint state
bridge.publish_joint_state("/robot/joint_states")

# 訂閱 cmd_vel (速度指令)
bridge.subscribe_cmd_vel("/robot/cmd_vel", callback)

# 發布相機影像
bridge.publish_camera_image("/camera/rgb", camera_prim)

ROS2 Action 整合

# Isaac Sim 可以作為 ROS2 Action Server
from omni.isaac.ros2_bridge import Ros2ActionServer

pick_server = Ros2ActionServer(
    "/robot/pick_object",
    PickAction,
    execute_callback=my_pick_callback

感測器模擬

相機 (RTX-based)

from omni.isaac.sensor import Camera

# 建立相機
camera = Camera(
    prim_path="/Robot/Camera",
    resolution=(1280, 720),
    frequency=30,  # Hz
    fov=90
)

# 讀取影像
rgb = camera.get_rgb()        # numpy array (720, 1280, 3)
depth = camera.get_depth()    # 深度圖
seg = camera.get_instance_segmentation()  # 語義分割
bboxes = camera.get_bounding_box_2d_tight()  # 2D bounding boxes

# 發布到 ROS
camera.publish_to_ros("/camera/rgb", "/camera/depth")

Software-in-the-Loop (SIL) 測試

什麼是 SIL？

傳統開發:  實體機器人 → 測試 → 失敗 → 硬體損壞
SIL開發:   虛擬機器人 → 測試 → 失敗 → 重來 (零成本)
            │
            通過測試後→ 部署到實體機器人

SIL 測試流程

Synthetic Data Generation

用模擬環境產生標註好的訓練資料：

from omni.isaac.synthetic import SyntheticData

sdg = SyntheticData()

# 設定輸出
sdg.set_output_path("/data/synthetic/")

# 隨機化物體位置、材質、燈光
sdg.randomize_objects()
sdg.randomize_materials()
sdg.randomize_lighting()

# 擷取標註資料
for i in range(1000):  # 產生 1000 張
    sdg.randomize_scene()
    rgb = camera.get_rgb()
    seg = camera.get_instance_segmentation()
    bboxes = camera.get_bounding_box_2d_tight()

    # 儲存 (image, segmentation mask, bounding boxes)
    sdg.save_frame(i, rgb, seg, bboxes)

完整範例：Pick-and-Place

from omni.isaac.core import World
from omni.isaac.core.robots import Robot
from omni.isaac.manipulators import PickPlaceController
from omni.isaac.core.objects import DynamicCuboid

# 初始化
world = World()
robot = world.scene.add(
    Robot(prim_path="/World/Franka", usd_path="Franka.usd")
)
cube = world.scene.add(
    DynamicCuboid(
        prim_path="/World/Cube",
        position=(0.5, 0, 0.05),
        size=0.05,
        color=(1, 0, 0)
    )
)

# Pick-and-Place 控制器
controller = PickPlaceController(
    name="pick_place",
    gripper=robot.gripper,
    robot_articulation=robot,
)

# 目標位置
controller.pick("/World/Cube")
controller.place(position=(0.3, 0.3, 0.05))

# 執行
while not controller.is_done():
    world.step(render=True)
