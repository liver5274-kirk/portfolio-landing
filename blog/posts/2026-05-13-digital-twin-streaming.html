什麼是數位孿生（Digital Twin）？

在 Omniverse 脈絡下，Digital Twin 的技術堆疊：

┌────────────────────────────────────┐
│      App Streaming (Web Client)    │  ← 使用者在瀏覽器中操作
├────────────────────────────────────┤
│      Kit Application               │  ← Omniverse 應用
│       ├── USD Scene Graph          │  ← 場景資料
│       ├── RTX Renderer             │  ← 即時光追渲染
│       └── Physics / Sensors        │  ← 物理模擬
├────────────────────────────────────┤
│      Omniverse Nucleus             │  ← 協作伺服器
├────────────────────────────────────┤
│      Data Connectors               │  ← IoT / CAD / 感測器
│      (PTC, Siemens, Autodesk...)   │
└────────────────────────────────────┘

Kit App Streaming 架構

核心概念

Kit App Streaming 讓 Omniverse 應用跑在雲端 GPU，透過 WebRTC 串流到瀏覽器：

[GPU 伺服器] ──WebRTC──→ [使用者瀏覽器]
    │                        │
  渲染 + 物理               鍵盤/滑鼠輸入
  USD 場景操作               低延遲互動

優勢

  • 不需要本機 RTX GPU
  • 瀏覽器即可操作（Chrome/Edge）
  • 適合企業內部部署
  • 支援多用戶同時檢視（類似 Google Docs）

部署模式

模式說明
Local Streaming本機 GPU 渲染，串流到同網路其他裝置
Cloud StreamingAWS/Azure 上的 GPU VM，WebRTC 串流
Omniverse CloudNVIDIA 託管（付費）

建立 Streaming App

Step 1: 建立 Kit App

# 建立 App 設定檔
mkdir my_digital_twin
cat > my_digital_twin/my_app.kit << 'EOF'
[settings]
app/exts/omni.kit.window.streaming.enabled = true
app/exts/omni.kit.window.streaming.port = 8111

# 啟用必要 Extension
[extensions]
omni.kit.window.viewport = {}
omni.kit.window.streaming = {}
omni.usd.schema.streaming = {}
EOF

Step 2: 啟動 Streaming Server

import omni.kit.streaming

# 啟動 WebRTC streaming
streaming = omni.kit.streaming.get_streaming_interface()
streaming.start_server(port=8111)

# 取得連線 URL
url = streaming.get_viewer_url()
print(f"Open in browser: {url}")

Step 3: 設定 Nucleus 協作

# extension.toml
[settings]
# Nucleus 伺服器位置
app/exts/omni.kit.window.streaming/nucleus_server = "omniverse://my-server.com"

# 允許多用戶編輯
app/exts/omni.kit.window.streaming/multi_user = true

App Streaming 效能考量

因素建議
GPURTX A5000 以上 (多用戶並發)
VRAM24GB+（複雜場景）
頻寬每用戶 10-25 Mbps（1080p）
延遲< 50ms（WebRTC）
最大用戶數依 GPU 效能，約 4-8/GPU

Streaming Settings Tuning

import omni.kit.streaming

streaming = omni.kit.streaming.get_streaming_interface()
streaming.set_quality_preset("balanced")  # low / balanced / high
streaming.set_max_fps(30)
streaming.set_resolution(1920, 1080)
streaming.set_bitrate(15000000)  # 15 Mbps
